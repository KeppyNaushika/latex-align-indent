# LaTeX Align

LaTeX表組み環境の自動整列拡張機能です。全角文字に対応しています。

## 機能

- `align`, `tabular`, `array` 環境の自動整列
- 全角文字の幅を正確に計算
- カーソル位置を保持

## 使用方法

1. LaTeXファイルを開く
2. `Ctrl+Shift+L` (Windows/Linux) または `Cmd+Shift+L` (Mac) を押す
3. または右クリックメニューから「LaTeX環境を整列」を選択

## 対応環境

- align / align*
- tabular / tabular*
- array / array*
- nodisplayskipflalign / nodisplayskipflalign*
